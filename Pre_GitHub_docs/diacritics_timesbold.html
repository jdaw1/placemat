/TimesNewRomanPS-BoldMT: accented characters and their PostScript glyphs

In the placemat software the default font is /TimesNewRomanPS-BoldMT, which contains many characters with accents and other diacritics. This document shows how to access these characters within PostScript.

Accented characters consist of a base character, and a diacritic, and the name of the glyph is the former followed by the latter. For example, /acircumflex is “â”, and /Acircumflex is “Â”. Observe the case sensitivity. So the PostScript compound string [(Ch) /acircumflex (teau L) /eacute (oville-Barton)] would be rendered as “Château Léoville-Barton”.
